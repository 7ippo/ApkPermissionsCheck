# ApkPermissionsCheck

## 1.apkPermissionsCheck

*To use:*

1. Set NECESSARYPERMISSIONSSET
2. python apkPermissionsCheck.py -path {apkpath}
3. Or: python apkPermissionsCheck.py -apkname 1.apk 2.apk ...
